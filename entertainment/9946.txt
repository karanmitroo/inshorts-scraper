Names of Bollywood celebrities, including Alia Bhatt, Katrina Kaif and Priyanka Chopra, are being used to refer to drugs by the underworld, as per the Mumbai Police. While Alia is code name for cocaine, Katrina is used to refer to smack while Priyanka's name is code for LSD. Further, Ranveer Singh's name is used to refer to a drug peddler.
